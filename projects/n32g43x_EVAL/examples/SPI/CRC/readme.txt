1、功能说明

	1、SPI 发送接收数据进行 CRC 校验


2、使用环境

	/* 硬件环境：工程对应的开发硬件平台 */
	开发板：NS3602_LQFP100_V0.1

3、使用说明
	
	/* 描述相关模块配置方法；例如:时钟，I/O等 */
	1、SystemClock：108MHz
	2、GPIO：SPI1: SCK-PA5   <--> SPI2: SCK-PB13
			 SPI1: MISO--PA6 <--> SPI2: MISO--PB14
			 SPI1: MOSI--PA7 <--> SPI2: MOSI--PB15


	/* 描述Demo的测试步骤和现象 */
	1.编译后下载程序复位运行；
	2.SPI1、SPI2 同时收发数据，传输完成后，发送 CRC 数据，检查数据和 CRC 值，查看 TransferStatus1 和 TransferStatus2 状态为 PASSED，
	再查看 CRC 值；
