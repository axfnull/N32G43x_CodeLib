1、功能说明

	1、SPI 读、写、擦除 W25Q128


2、使用环境

	/* 硬件环境：工程对应的开发硬件平台 */
	开发板：NS3602_LQFP100_V0.1 和 N32G457QE_EVB V1.0（NS3602_LQFP100_V0.1 开发板上没有 W25Q128）

3、使用说明
	
	/* 描述相关模块配置方法；例如:时钟，I/O等 */
	1、SystemClock：108MHz
	2、SPI1: NSS--PA4、SCK--PA5、MISO--PA6、MOSI--PA7


	/* 描述Demo的测试步骤和现象 */
	1.编译后下载程序复位运行；
	2.通过 SPI1 读取 W25Q128 的 ID，然后写数据，再读出来，比较读写数据，查看 TransferStatus1 状态为 PASSED，
	  然后擦除块，检查擦除块正常。
