	/* 简单描述工程功能 */
        这个例程配置并演示使用DMA将SRAM数据写入FLASH


2、使用环境

	/* 硬件环境：工程对应的开发硬件平台 */
        开发板：N32G43XM-STBV1.0


3、使用说明
	
	/* 描述相关模块配置方法；例如:时钟，I/O等 */
        SystemClock：108MHz

        USART：TX - PA9，RX - PA10，波特率115200
	DMA：数据传输方向为SRAM到FLASH

	/* 描述Demo的测试步骤和现象 */
        1.编译后下载程序复位运行；
        2.查看串口打印信息，对比传输完成后的SRAM及FLASH数组数据，完全相同则打印信息为测试通过，程序运行正常；
