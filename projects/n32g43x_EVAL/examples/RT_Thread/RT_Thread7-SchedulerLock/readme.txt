1、功能说明

	1、此例程展示在 RT_Thread 系统调用调度器锁

2、使用环境

	/* 硬件环境：工程对应的开发硬件平台 */
	开发板：N32G43XR-STB V1.0

3、使用说明
	
	/* 描述相关模块配置方法；例如:时钟，I/O等 */
	SystemClock：108MHz
	GPIO:	LED:D2--PB4、D3--PB5
	日志：串口打印 PA9(TX)，波特率：115200

	/* 描述Demo的测试步骤和现象 */
	1.编译后下载程序复位运行
	2.接好串口打印工具
	2.本例程创建三个线程，开始线程、LED0 线程和 LED1 线程，开始线程用于创建其他线程（LED0和LED1），LED1 线程优先级高于 LED0, LED0 线程将调度器上锁后，在解锁之前，不会切换到 LED1 线程，查看打印日志

	无
