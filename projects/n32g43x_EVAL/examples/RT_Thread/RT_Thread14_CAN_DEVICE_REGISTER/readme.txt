1、功能说明

	1、此例程展示在 RT_Thread 系统创建 CAN 设备

2、使用环境

	/* 硬件环境：工程对应的开发硬件平台 */
	开发板：N32G43XR-STB V1.0

3、使用说明
	
	/* 描述相关模块配置方法；例如:时钟，I/O等 */
	SystemClock：108MHz
	GPIO:	LED:D2--PB4、D3--PB5
            KEY:S1--PA4(KEY1),S2--PA5(KEY2)
    CAN:    RX--PA11、TX--PA12
    串口:   TX--PA9

	/* 描述Demo的测试步骤和现象 */
	1.编译后下载程序复位运行
	2.本例程创建三个线程，开始线程、LED0 线程和 LED1 线程，开始线程用于创建其他线程（LED0和LED1），LED0 线程用于控制 D2 500ms闪烁，LED1 线程读取 KEY1 电平，高则 D3 亮，低则 D3 灭，KEY2 按下触发中断，并在回调函数中打印日志。
      在开始线程测试CAN设备，用CAN卡接上RX和TX，在开始线程发送CAN数据，可用CAN卡在上位机上查看发送的数据，发送完以后，由于是环回模式，发送完以后回接收到刚发送的数据，判断接收数据是否正常，用串口打印测试结果
