1、功能说明

	1、此例程展示在 RT_Thread 系统创建 SPI 设备，并读写操作 W25Q128
    
2、使用环境

	/* 硬件环境：工程对应的开发硬件平台 */
	开发板：N32G43XR-STB V1.0  N32G457QEL_EVB V1.1

3、使用说明
	
	/* 描述相关模块配置方法；例如:时钟，I/O等 */
	SystemClock：108MHz
	GPIO:  SPI1: NSS--PA4、SCK--PA5、MISO--PA6、MOSI--PA7
    串口： TX--PA9


	/* 描述Demo的测试步骤和现象 */
	1.编译后下载程序复位运行
	2.本例程创建三个线程，开始线程、LED0 线程和 LED1 线程，开始线程创建其他线程（LED0和LED1），LED0 线程用于控制 D2 1s闪烁，LED1 线程用于控制 D3 500ms闪烁，
      在开始线程中读写操作W25Q128，测试结果通过串口打印
    

4、注意事项
