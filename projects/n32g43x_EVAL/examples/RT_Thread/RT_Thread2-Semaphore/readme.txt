1、功能说明

	1、此例程展示在 RT_Thread 信号量的创建、获取和释放

2、使用环境

	/* 硬件环境：工程对应的开发硬件平台 */
	开发板：N32G43XR-STB V1.0

3、使用说明
	
	/* 描述相关模块配置方法；例如:时钟，I/O等 */
	SystemClock：108MHz
	GPIO：PB5 控制 LED(D2) 亮灭；PB4 控制 LED(D2) 闪烁
	            KEY：S1--PA4（KEY0）

	/* 描述Demo的测试步骤和现象 */
	1.编译后下载程序复位运行
	2.本例程创建四个线程，开始线程、LED0 线程、 LED1 线程 和 KEY 线程，开始线程用于创建其他线程（LED0、LED1 和 KEY），LED0 线程用于控制 D2 亮灭，LED1 线程用于控制 D3 500ms闪烁，
	  KEY 线程扫描按键，当 KEY0 检测到按下时，释放信号量，LED0 线程获取信号量，获取成功后，翻转 D2；
